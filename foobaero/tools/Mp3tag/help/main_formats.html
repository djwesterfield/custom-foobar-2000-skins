Supported filetypes

  • Advanced Audio Coding (AAC)
  • Monkey's Audio (APE)
  • Free Lossless Audio Codec (FLAC)
  • MPEG Audio Layer 3 (MP3)
  • MPEG-4 (iTunes® compatible MP4 / M4A / M4B)
  • Musepack (MPC)
  • Ogg Vorbis (OGG)
  • OptimFROG/OptimFROG DualStream (OFR/OFS)
  • Speex (SPX)
  • True Audio (TTA)
  • Windows Media Audio (WMA)
  • WavPack (WV)
