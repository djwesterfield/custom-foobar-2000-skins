Tags

Save tags when using arrow-keys/single mouse click

This option enables automatic saving of entered data when the file loses the focus.

Preserve timestamps of modified files

Mp3tag will keep the modification date of the files, when writing or removing tags.

Restrict incoming files to

Mp3tag will read all supported files with the extensions given at Restrict incoming files to.

Save mode at extended tag dialog

You can specify whether Mp3tag should ask, ignore or always save changes made at the extended tag dialog.

Don't display first image from file directory as cover art

Normally, Mp3tag displays the first image from the files' directory as cover art in the Tag Panel if the file has no embedded cover art. You can restrict cover display to embedded cover art with this option.
