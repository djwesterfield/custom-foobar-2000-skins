Configuration > Genres

User defined Genres

Here you can add some extra user defined genres to the standard list of genres. These genres are not saved to an ID3v1 tag due to the limitations of the tag format.

Show only user defined genres


If this option is enabled, the genre dropdown list at the tag input mask only shows user defined genres after a restart of Mp3tag.
